Solution requires minikube or docker-for-desktop and helm

Install: Helm https://helm.sh/docs/intro/install/
Install: Dapr CLI https://docs.dapr.io/getting-started/install-dapr-cli/
Install: kubectl https://kubernetes.io/docs/tasks/tools/

1) Run: $ dapr init -k

2) Go to: deploy\helm 
	Run: $ helm install payments-test .

3) Teardown after tests
	Run: $ helm uninstall payments-test


- In case you want to rebuild containers use 
docker build -f src\BankProxy\BankProxy.API\Dockerfile -t kapitan123/bank-proxy.api .
docker push kapitan123/bank-proxy.api

docker build -f src\MerchantPayment\MerchantPayment.API\Dockerfile -t kapitan123/merchant-payment.api .
